Homework 3¶

Most people are unaware how many births are actually taking place in the US each year, month, or day. Luckily, we have access to actual data! This homework assignment will give you the chance to demonstrate your understanding and mastery of the following:

  1. Reading data from files
  2. Using loops and condititional statements to filter data
  3. Plotting relevant information

This is an oppurtunity to make some nice looking plots that reveal something interesting about the underlying US birth data.

The data set us_births_2000_2014.csv contains the number of births in the US for the years 2000-2014. Each row in the file corresponds to a single day. The Columns of the csv file are as follows:

  1. year: The year of the particulr day (2000-2014)
  2. month: The month of the particular day (1-12)
  3. date_of_month: The numerical date of the particular day (1-31)

  4. day_of_week: The day of the week for the day

    1=Monday, 2=Tuesday, 3=Wednesday, 4=Thursday, 5=Friday, 6=Saturday, 7=Sunday

  5. births: The total number of births recorded in the US for that day

The data for this assignment is on Moodle and on the jupyterhub server at /srv/data/hw3/us_births_2000_2014.csv. The data is taken from FiveThirtyEight.com.

Assignment¶

You need to answer the following 3 questions with plots and words:

  1. What is the total number of births for each year?
    • X axis should be year, Y axis should be total births.
  1. What is the most common day of the week for births (independent of year)?
    • X axis should be day of week (Monday, etc.), Y axis should be number of births.
  1. Does this day of the week change over the 15 years of the data?
    • X axis should be day of week (Monday, etc.), Y axis should be number of births.
    • Different lines/bars/etc for each year, all on the same plot

For each question, give a plot and a few sentences (in a separate markdown cell) that explains what the plot shows and how we can interpret it (i.e., the actual answer to the question).

Approach¶

As always, the best path to answering the above questions is to decompose this assignment into smaller parts. If you can solve several smaller problems, then you will be well on your way to solving the big problems. As a place to start, here are some tips:

import matplotlib.pyplot as plt

def birth_analysis():
    # This function opens and reads the file, then applies the data to dictionaries based on the problems given. The
    # dictionary births has the year as the key, and the total number of births as the value. The dictionary day_of_week 
    # has the weekday as the key, and the total number of births given in the data for that year. The dictionary dow_year
    # has the year as the key, and then a list of the number of births each day for that given year. 
    open_file = open("/srv/data/hw3/us_births_2000_2014.csv")
    header = open_file.readline().strip()
    birth_data = open_file.readlines()
    
    births = {}      # <--- Problem 0
    day_of_week = {} # <--- Problem 1
    dow_year = {}    # <--- Problem 2
    
    for line in birth_data:
        line = line.strip().split(",")

        year = line[0]
        week_day = line[3]
        num_births = line[4]

        if year not in births.keys():
            births[year] = int(num_births)
        else:
            births[year] += int(num_births)

        if week_day not in day_of_week.keys():
            day_of_week[week_day] = int(num_births)
        else:
            day_of_week[week_day] += int(num_births)

        if year not in dow_year:
            dow_year[year] = [0]*7
            dow_year[year][int(week_day)-1] = int(num_births)
        else:
            dow_year[year][int(week_day)-1] += int(num_births)

    open_file.close()
    return births, day_of_week, dow_year

def list_sorter(key, unsorted_list):
    # This function sorts lists and keys so that they are in useable order for graphing. 
    sorted_key_indexes = [key.index(i) for i in sorted(key)]
    output = []
    for n, i in enumerate(unsorted_list):
        output.append(unsorted_list[sorted_key_indexes[n]])
    return output

def graph_birth_analysis():

    births, day_of_week, dow_year = birth_analysis()
    
    year_list = list(births.keys())
    births_year_list = list(births.values())
    days_list = ['Monday','Tuesday','Wednesday','Thursday','Friday','Saturday','Sunday']
    days_list_2 = ['Saturday','Sunday','Monday','Tuesday','Wednesday','Thursday','Friday']
    births_day_list = list(day_of_week.values())
    births_day_year_list = list_sorter(list(dow_year.keys()), list(dow_year.values()))
    
    # Problem 0 - Graphs Number of births per year
    plt.figure(dpi = 200, figsize = (10,5))
    plt.bar(range(15), births_year_list, color = "cornflowerblue")
    plt.xticks(range(15), year_list, rotation = 90)
    plt.ylim(3.5e6,4.5e6)
    plt.ylabel("Millions of Births")
    plt.xlabel("Year")
    plt.title("# of Births per year (2000-2014)")
    
    # Problem 1 - Graphs Number of births by day of the week
    plt.figure(dpi = 200, figsize = (10,5))
    plt.bar(range(7), births_day_list, color = "cornflowerblue")

    plt.ylim(0.5e7, 1.1e7)
    plt.ylabel("Tens of Millions of Births")
    plt.xlabel("Days of the Week")
    plt.title("Most Common Day for Births (Independent of Year)")
    
    # Problem 2 - Graphs clusters of points representing the number of births each year for that day of the week
    plt.figure(dpi = 200, figsize = (10,5))
    for n, i in enumerate(births_day_year_list):
        color_list = ["blue", "red", "green", "purple", "aqua", "gold", "black", "magenta", "mediumseagreen", "midnightblue", "cornflowerblue", "orange", "olivedrab", "salmon", "tomato"]

    plt.ylim(3.5e5, 7.5e5)
    plt.xticks(range(7), days_list, rotation = 90)
    plt.xlabel("Day of Week")
    plt.ylabel("Births")
    plt.title("Births per Year by Day")
    return
